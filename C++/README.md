# C++ Implementation

## Requirements
 * CMake >= 3.0
 * C++2a
    * GCC >= 9.0
    * Clang >= 9.0

## build
Use `cmake` to generate make files from C++/CMakeLists.txt. Use `make all` from the same directory as the `cmake` call to generate `DistributedDB` executable file.

ex. \
From Distributed-DB/C++
```
mkdir build
cd build
cmake ..
make all
```
